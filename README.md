# College Admission Management System

A comprehensive web-based college admission management system built with HTML, CSS, and JavaScript.

## 🚀 Features

### 🏠 Homepage
- College information and navigation
- Course offerings display
- Responsive design

### 📝 Admission Form
- Student registration form
- Form validation (Name, DOB, Marks, Category, Course)
- Data storage in localStorage

### 📊 Cutoff Marks
- Display course-wise cutoff marks
- Category-based cutoffs (General, BC, MBC, SC)

### 🏆 Merit List
- Filter students based on cutoff marks
- Display selected/rejected status
- Statistics dashboard

### 🔐 Admin Panel
- Secure login system
- Cutoff calculation tool
- Formula: Cutoff = (Maths/2) + (Physics/4) + (Chemistry/4)

## 🛠️ Technologies Used

- **HTML5**: Semantic markup and structure
- **CSS3**: Modern responsive design with Flexbox/Grid
- **JavaScript**: Form validation, localStorage, dynamic content
- **localStorage**: Client-side data persistence

## 📱 Responsive Design

The application is fully responsive and works seamlessly across:
- Desktop computers
- Tablets
- Mobile devices

## 🎯 How to Use

### For Students:
1. Visit the homepage to learn about courses
2. Fill out the admission form
3. Check merit list for selection status

### For Administrators:
1. Login to admin panel
2. Calculate cutoffs using the formula
3. View and manage admission data

## 📊 Cutoff Calculation Formula

```
Cutoff = (Maths Marks ÷ 2) + (Physics Marks ÷ 4) + (Chemistry Marks ÷ 4)
```
